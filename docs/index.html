Claude Env Check

Audit Claude Code proxy, timezone, DNS, IPv6, TUN, and environment-variable residue before they turn into confusing routing or fingerprint problems.

Install

claude-env-check
claude-env-check --offline

What It Checks

Timezone

Compares macOS and Node.js timezone surfaces and flags China timezone mismatch risks.

Proxy Visibility

Reads macOS system proxy state and common proxy environment variables without changing them.

Network Exit

Checks IPv4, IPv6, DNS heuristics, TUN hints, and browser retest links for final confirmation.

Use With Codex

After installation, ask your agent:

Use $claude-env-check to audit my Claude Code proxy, timezone, DNS, and environment-variable risk.

Interpreting Results

FAILHigh-risk signal that should be handled before relying on the environment.
WARNSuspicious residue or mismatch that needs human review.
PASSNo issue found for that specific local check.
